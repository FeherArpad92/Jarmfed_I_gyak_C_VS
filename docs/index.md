# Járműfedélzeti Rendszerek I. - Gyakorlat

Üdvözöllek a tárgy gyakorlati segédanyagait tartalmazó oldalon. Itt találod a heti gyakorlatok kódjait, magyarázatait és a kapcsolódó feladatokat.

A bal oldali menüben választhatod ki az aktuális hetet. A kódok közvetlenül a GitHub repozitóriumból származnak, de itt kiegészítő magyarázatokkal látjuk el őket a mélyebb megértés érdekében.
